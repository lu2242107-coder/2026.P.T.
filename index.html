VIP 稅務與傳承

1. 所得/海外/不動產
個人 CFC 所得申報 +
【核心規範】CFC盈餘達700萬須申報。若信託孳息受益人比例未明定，須按「人數平均」推算受益比例申報，以免漏報受罰。
📂 案例說明：
甲將BVI公司100%股權交付信託，約定4名子女為受益人並依「績效」分配。當年CFC盈餘9000萬，大兒子績效普通僅實拿20%(1800萬)，但因合約未定死比例，報稅時須依法按4人平分(25%)，申報2250萬CFC營利所得，多繳稅也無法退稅 [1, 2]。
🔗 查閱 CFC 信託實務補稅案例
海外房產資金回台 +
【核心規範】處分海外房產資金匯回，須申報財產交易所得。若無法提出原始取得成本證明，依法按成交價額之 12% 計算所得額。
📂 案例說明：
陳小姐於2005年購入香港房產，2024年以新台幣4960萬出售並匯回台灣。因年代久遠單據遺失，國稅局按12%計算所得為595.2萬，併入最低稅負制計算基本稅額。若事後被查獲有更低成本的單據卻未報，將面臨重罰 [3]。

外籍人士不動產課稅 +
【核心規範】外籍人士買賣台灣不動產須符合「互惠原則」。適用房地合一稅2.0，境外法人短期持有(2年內)出售稅率高達45%。
📂 案例說明：
外資全資設立開曼公司，花1億元購入台北商辦，持有一年半後以3億元售出。因屬境外法人且持有未滿2年，直接面臨45%重稅（需繳近9000萬）。若當初設立「國內投資公司」間接持有並長期規劃，整體稅負將大幅降低 [4]。

舊制自用住宅重購退稅 +
【核心規範】納稅義務人列報扶養之未成年子女出售自用住宅，自移轉日起2年內由「同一」子女重購較高價的自用住宅，得適用重購退抵稅額。
📂 案例說明：
父親將名下舊制公寓贈與16歲兒子。兒子隔年出售該公寓，並於1年半後以自己名義買下另一間總價更高的透天厝自住。因均屬「同一未成年子女」名下，父親申報綜所稅時，可依法扣抵或退還該筆財產交易所得稅 [5]。

2. 扣除/申報
免稅額與扣除額調高 +
【核心規範】114年度基本生活費調高至21.3萬；長照扣除額調高至18萬；租金支出改列特別扣除額；幼兒扣除額擴大至6歲且取消排富。
📂 案例說明：
甲君為單薪家庭，扶養2名大學生與2名父母（共6人）。依114年新制，全戶基本生活費為127.8萬。扣除免稅額與標準扣除額後，甲君可額外獲得37.4萬的「基本生活費差額」自所得總額中減除，大幅降低應納稅額 [6, 7]。

綜所稅申報 SOP +
【核心規範】單身所得低於22.8萬可免申報。手機報稅4.0支援編修功能，附件上傳總容量提升至15MB。
📂 案例說明：
社會新鮮人小林去年薪資僅20萬，依法可免辦結算申報。而台商王總因醫療與捐贈收據繁多，今年使用「手機報稅4.0」，直接用手機拍照上傳高達15MB的附件並修改扣除額，無須再親跑國稅局遞交紙本單據 [8]。

稅務新時代關鍵議題
高爾夫球會員證稅務 +
【核心規範】高爾夫球會員證屬財產交易，無論出售、贈與或繼承，均須以「市價」為基準申報，勿以公司帳面淨值低估。
📂 案例說明：
甲先生將某育樂公司股票（兼具高爾夫球會員資格）贈與兒子。甲先生以公司「每股淨值60萬」申報，但國稅局查出該會員證市場行情高達320萬，認定屬不相當代價移轉，當場補徵7.6萬贈與稅並加計利息 [9]。

頂客與單身族遺產繼承 +
【核心規範】單身族若無子女與父母，遺產將100%由兄弟姊妹繼承。若想留給非親屬，需透過保險或信託對抗「特留分」。
📂 案例說明：
某知名單身資深藝人猝逝，生前曾公開表示不願將上億遺產留給疏遠的兄長，想全給照顧她的乾兒子。但因未妥善規劃，依法兄長仍能主張「特留分」強勢分產。若生前將乾兒子列為人壽保險受益人，便能合法隔離這筆資產 [10]。

國內生前信託課稅 +
【核心規範】生前信託可達到資產隔離。若為「他益信託」且不保留變更權，設立時即課贈與稅，未來獲利課受益人所得稅。
📂 案例說明：
某富商擔心自己猝逝後，龐大遺產會被離異的前妻以「未成年子女監護人」身分掌控揮霍。他生前設立了「他益信託」，約定資產由銀行代管，待子女滿25歲後分期領取。此舉不僅保障了子女生活，也成功隔離了前妻的干預 [11]。

扶養親屬列報實務 +
【核心規範】同一直系尊親屬僅能由一人列報扶養。若兄弟姊妹重複申報且無協議，國稅局將依「實際扶養事實」認定。
📂 案例說明：
兄弟兩人都把母親報在自己的綜所稅裡搶免稅額。由於兩人不願妥協，國稅局介入調查「實際扶養事實」。經查明，母親實際與弟弟同住滿300天，且醫療與長照費用均由弟弟帳戶扣款，最終判定免稅額歸屬弟弟，哥哥遭剔除補稅 [12]。

家族傳承安排策略
遺產稅實物抵繳 +
【核心規範】應納遺產稅達30萬且「現金繳納有困難」得申請實物抵繳。若被繼承人遺留現金大於稅款，國稅局將駁回抵繳。
📂 案例說明：
老董事長過世，遺產含1000萬土地與2億元存款，應納稅額為3211萬。繼承人想用土地「實物抵繳」保留現金。但國稅局認定帳戶有2億現金「並無繳現困難」，直接駁回申請。若存款僅500萬，則可合法用土地或股票抵繳 [13]。
🔗 查閱遺產稅實物抵繳遭拒判例
擬制遺產課稅 +
【核心規範】死亡前2年內贈與配偶財產應併計遺產。依114年新制，該財產視為現存，配偶仍可主張「剩餘財產差額分配請求權」。
📂 案例說明：
好野君過世前半年匯款5億元給妻子，過世時自己帳戶僅剩1.2億。這5億依法被國稅局「擬制」拉回算遺產。但依據憲法法庭判決與最新處理原則，妻子仍可將這5億列入計算，行使剩餘財產分配請求權，大幅抵減了天價遺產稅 [14]。

全球設籍稅務注意 +
【核心規範】家族二代若取得雙重國籍或長居海外，將面臨 CRS 資訊交換透明化，且跨國繼承時極易面臨兩國雙重課徵遺產稅。
📂 案例說明：
富豪的兒子早年取得美國綠卡並定居加州。富豪在台灣過世後，兒子不僅要繳納台灣遺產稅，因其具備美國稅務居民身分，這筆龐大遺產還必須向美國 IRS 申報。由於台灣能扣抵的海外稅額設有上限，最終導致嚴重的雙重課稅損失 [15]。

代際傳承保險工具 +
【核心規範】保險指定受益人原則免計遺產。高資產族常規劃「三代保單」，達到傳承、預留稅源與免計基本所得的三重效益。
📂 案例說明：
爺爺規劃「三代保單」：自己出錢(要保人)，以兒子為被保險人，孫子為身故受益人。當兒子過世時，孫子獲賠上千萬保險金。依規此理賠金不計入兒子的遺產；且只要全戶死亡給付低於3740萬，孫子連基本所得稅都不用繳，完美傳承 [16]。

資產保全與企業永續 +
【核心規範】為防後代爭產與股權外流，應將信託、閉鎖性公司、保險與遺囑等多管齊下，築起不可撼動的傳承防護網。
📂 案例說明：
某老牌製造廠為防二代爭產賣股，成立「閉鎖性公司」持有集團所有股權，章程嚴格限制股份轉讓，並發行擁有「黃金否決權」的特別股給接班長子。同時輔以家族憲章，確保家族資產不因婚姻或個人負債而外流，實現永續經營 [17]。

家族辦公室全解析 +
【核心規範】家族辦公室面臨專業人才短缺，應從「獵才」轉向「育才」，並賦予專業經理人「共同投資權」以深化歸屬感。
📂 案例說明：
傳產大亨二代無意接班，大亨遂設立「單一家族辦公室(SFO)」，聘請頂尖投資長代管百億資金。為防高階人才跳槽，大亨在治理機制中加入「共同投資權」，讓投資長以自有資金與家族按比例入股標的，成功把外人變為利益共同的家人 [18, 19]。
